Scrivere un programma in linguaggio C che:
• legga dal file “input.txt” le dimensioni e i valori di una matrice di interi
• eliminare dalla matrice tutte le righe che sono uguali al loro inverso (palindrome)
• salvare ogni colonna rimasta della matrice in un vettore di struct definito come segue
#define N_MAX 100
struct Mystruct
{
int indice;
int n; 
int valori[N_MAX];
float media; 
};
Dove:
indice è l’indice della colonna
n è il numero di elementi della colonna
valori sono gli elementi della colonna
media è la media degli elementi della colonna
• Ordinare il vettore per valori di media decrescenti
• Salvare il vettore di struct in un file binario “output.bin”
Esempio
4 5
1 2 3 4 5
1 1 3 2 2
8 6 4 2 0
1 2 3 2 1 
Matrice finale:
1 2 3 4 5
1 1 3 2 2
8 6 4 2 0
Array di struct:
{0, 3, 3.33 [1 1 8 ]}
{2, 3, 3.33 [3 3 4 ]}
{1, 3, 3.00 [2 1 6 ]}
{3, 3, 2.67 [4 2 2 ]}
{4, 3, 2.33 [5 2 0 ]}
