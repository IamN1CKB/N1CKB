Scrivere un programma in linguaggio C che:
• legga dal file “input.txt” la dimensione e i valori di una matrice quadrata di caratteri
• scorrendo tutte le righe e le colonne della matrice individuare tutte le parole presenti in 
verticale o orizzontale, ovvero tutte le sequenze di caratteri diverse da uno spazio, ma più 
lunghe di una lettera
• salvare ogni parola in una struct definita come segue
#define N_MAX 100
struct Mystruct {
    int inizio[2];
    int fine[2];
    char parola[N_MAX];
};
Dove:
inizio contiene gli indici della matrice corrispondenti alla posizione della 
prima lettera della parola trovata
fine contiene gli indici della matrice corrispondenti alla posizione dell’ultima 
lettera della parola trovata
parola è la parola trovata
• Ordinare il vettore per valori alfabetici crescenti del campo parola
• Salvare il vettore di struct in un file binario “output.bin”
Esempio

| A       D         |
| S A R T O R I A   |
| A   A   R         |
|     P A N I N O   |
| R   A   A         |
| O       T   B O A |
| S O S T A         |
| A         B A R   |
